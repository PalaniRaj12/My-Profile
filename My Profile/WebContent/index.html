A Fews words about Gamification String

Gamification String is a Coding Game where player gets points by solving all the test cases for String based problem posted by Admin. Run button will compile and run the user's Java code. Outputs and Errors will be displayed on Screen separately. If the output of user code is matching with expected result status shows correct. Submit button checks all the test cases works correctly or not and shows points for the problem. User has option to view input string, expected result and resultant string for all testcases. Admin can edit or view the existing problems and can view user's points for each problem.

Front-End: HTML, CSS, JavaScript, JQuery

Back-End: Spring MVC, Hibernate, Oracle DataBase
